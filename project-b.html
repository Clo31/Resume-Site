Java Bookstore Management Application

Coursework • Solo

Description

Developed a console-based Java program to create and manage BookstoreBook objects (author, title, ISBN, price, sale status). Applied object-oriented principles (constructors, getters/setters), stored data dynamically in an ArrayList, and strengthened proficiency in user input handling, data encapsulation, and the Java standard library.
