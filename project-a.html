Tiny Compiler for a PL/0-Like Language

Coursework • Solo

Description

Created a tiny compiler in C for a PL/0-like language, featuring a custom lexer, recursive-descent parser, symbol table, and a code generator targeting a stack-based virtual machine. Demonstrated end-to-end compilation from lexical analysis to pseudo-assembly output, strengthening knowledge of compiler design, parsing theory, and data-structure management.

Skills Learned

  • Compiler Design
  • Parsing (Recursive-Descent)
  • Symbol Tables
  • Stack-based VM / Codegen
  • C Programming
  • Data Structures
